Documentation for TTTS (1.0.1)

Game Modes:

  • Standard

  • Classic

  • Random coefficient

  • Large area

  • Time limit

Standard

The standard mode is tic-tac-toe, but you get points for combinations. As in other modes, you have the ability to choose the number of rounds.

Move options

Classic

Classic mode is the usual tic-tac-toe, but with the ability to choose the number of rounds.

Random coefficient

The mode Random odds is a tic-tac-toe with combinations from the standard Move options mode, but here there is a random coefficient by which your points are multiplied at the end of the round (thus, if you are lucky, you can overtake your opponent in points) as in other modes you can choose the number of rounds. (Questionable) You can also limit the maximum and minimum odds .

Large area

Large area mode is represented by tic-tac-toe with combinations from the standard Move options mode but with the ability to select an area larger than the standard 3x3. As in other modes, you can choose the number of rounds.

Time limit

The Time limit mode is represented by tic-tac-toe with combinations from the standard Move options mode but with a limited time per move (For failure, a player is deducted 1 point every second of thought / The player loses), the time can be set in the settings of a new match. As in other modes, you can choose the number of rounds.

Документация для TTTS (1.0.1)

Игровые режимы:

  • Стандарт

  • Классический

  • Случайный коэффициент

  • Большая территория

  • Временной лимит

Стандарт

Стандартый режим представляет собой крестики нолики но за комбинации вы получаете очки.Как и вдругих режимах у вас есть возможность выбирать количество раундов.

Move options

Классический

Класический режим представляет собой обычные крестики нолики но с возможность выбрать количество раундов.

Случайный коэффициент

Режим Рандомный коэффициент представляет собой крестики нолики с комбинациями из стандартного режима Move options но здесь существует рандомный коэффициент на который множестся ваши очки в конце раунда (тем самым если вам повезёт то вы можете обогнать соперника по очкам) как и вдругих режимах в можете выбирать количество раундов. (Под вопросом) Так же вы можете ограничеть максимальный и минимальный коэффициент.

Большая территория

Режим Большая территория представляет собой крестики нолики с комбинациями из стандартного режима Move options но с возможностью выбрать территорию больше чем стандартный 3x3. Как и вдругих режимах в можете выбирать количество раундовю.

Временной лимит

Режим Временной лимит редставляет собой крестики нолики с комбинациями из стандартного режима Move options но с ограниченым временем на ход ( За провал с игрока снимаются 1 поинт каждую секунду раздумий / Игрок проигрывает ), время можно выставлять в настройках нового матча. Как и вдругих режимах в можете выбирать количество раундовю.
