Settings:

Auto-Round

Selective setting that allows you to automatically end the round when the players' actions end

My turn

Selective setting where each new round starts a different player. Round 1 (X) → Round 2 (0) → Round 3 (X) → Round 4 (0), etc.

Gamemodes:

Standard

The standard mode is tic-tac-toe, but you get points for combinations. As in other modes, you have the ability to choose the number of rounds.

Classic

Classic mode is the usual tic-tac-toe, but with the ability to choose the number of rounds.

Random coefficient

The Random coefficient mode is represented by tic-tac-toe with combinations from the standard Move-options mode, but here there is a random coefficient by which your points are multiplied at the end of the round, as in other modes, you can choose the number of rounds.

Large area

Large area mode is represented by tic-tac-toe with combinations from the standard Move-options mode but with the ability to select an area larger than the standard 3x3. As in other modes, you can choose the number of rounds.

Time limit

The Time limit mode is represented by tic-tac-toe with combinations from the standard Move-options mode, but with a limited time for the whole game, the time can be set in the settings of a new match. As in other modes, you can choose the number of rounds.
